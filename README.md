# Git Commit Tags

| Tag        | Description |
|------------|-------------|
| `Feat:`    | 새로운 기능 추가 |
| `Fix:`     | 버그 수정 |
| `Docs:`    | 문서 수정 |
| `Style:`   | 코드 포맷팅, 세미콜론 누락, 코드 변경 없음 |
| `Refactor:`| 코드 리팩토링 |
| `Test:`    | 테스트 코드 추가, 수정, 삭제 (비즈니스 로직 변경 없음) |
| `Chore:`   | 기타 변경사항 (빌드 스크립트 수정, assets, 패키지 매니저 등) |
| `Comment:` | 필요한 주석 추가 및 변경 |
| `Init:`    | 프로젝트 초기 생성 |
| `Rename:`  | 파일/폴더명 수정 또는 이동 |
| `Remove:`  | 파일 삭제 |
